# My Daily Time & Goals Tracker

A single page application that visualises daily activities and goals completion using a bar chart and dynamic goals list. Users can upload a JSON file and the dashboard automatically updates to display time spent on each activity and how many goals have been completed.

## How to Use the Application
1. Click the **Upload** button.
2. Select a JSON file containing activities and goals.
3. The bar chart will update to show time spent on each activity.
4. The goals list will update with a tick or a cross depending on completion.
5. Click the **Reset** button to clear the dashboard and upload a new file.

## Project Structure
- **index.html** - Main structure of the webpage
- **style.css** - Styling for layout, colours and formatting
- **script.js** - Handles JSON loading chart updates and goal logic
- **data/** - Contains the JSON file used for uploading into the application
- **README.md** - Documentation explaining the project

## Features
- Upload a JSON file to populate the dashboard
- Bar chart showing time spent on each activity
- Colour-coded activities
- Dynamic goals list with tick and cross symbols
- Summary of completed goals
- Reset button to clear the dashboard

## Technologies Used
- **HTML** - page structure
- **CSS** - layout and styling
- **JavaScript** - logic and interactivity
- **Chart.js** - bar chart visualisation

This project demonstrates:
